## 项目介绍

项目名: 海贼王乌索普.

简介: 探索和学习数据结构.

目前包含:

  - 动态数组 DynamicArrayV2 (可用于实现栈)
  - 循环数组 LoopArrayV3 (可用于实现双向队列)
  - 头指针链表 LinkedListV1 (可用于实现栈)
  - 头尾指针链表 LinkedListV2 (可用于实现队列)
  - 二分搜索树 BST (可用于实现集合, 字典)
  - 最大堆 MaxHeap (可用于实现优先级队列)
  - 线段树 SegmentTree
  - 字典树 Trie
  - 并查集 UnionFindV2
  - AVL 树 AVL (可用于实现集合, 字典)
